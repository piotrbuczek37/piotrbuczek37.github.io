February 2021 update: Private estimation sessions

NEW RELEASE Released two days ago

This month we present the ability to create several sessions of one type per board. For now, Relative and Bucket Sizing sessions are supported with Interactive and Asynchronous sessions support coming shortly.

The new release includes:

  • support of multiple estimation sessions of Relative and Bucket Sizing types;
  • quick join option for the 3 latest sessions;
  • all-sessions page with active and finished sessions list and sessions filtering options.

Other updates:

  • improved estimation field selection for Kanban boards;
  • improved session sharing via links for projects with multiple boards;
  • discontinued support of old UI;
  • performance, stability, and security improvements.
